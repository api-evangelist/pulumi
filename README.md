# Pulumi (pulumi)
Pulumi is a modern infrastructure as code platform that allows you to use  familiar programming languages to build, deploy, and manage cloud infrastructure. The Pulumi Cloud API enables programmatic access to manage organizations,  stacks, deployments, and other Pulumi Cloud resources.
